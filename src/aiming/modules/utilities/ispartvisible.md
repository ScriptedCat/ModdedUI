# Utilities.IsPartVisible
Returns whether a part is visible to the `CurrentCamera`

## Parameters
| Name             | Description | Type       | Default |
| ---------------- | ----------- | ---------- | ------- |
| `Part`           | N/A         | `Instance` | N/A     |
| `PartDescendant` | N/A         | `Instance` | N/A     |

## Return
`<boolean>` IsVisible
